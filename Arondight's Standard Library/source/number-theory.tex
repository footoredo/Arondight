\section{原根相关}
	\begin{enumerate}
		\item 模$m$有原根的充要条件：$m = 2, 4, p^a, 2p^a$，其中$p$是奇素数；
		\item 求任意数$p$原根的方法：对$\phi(p)$因式分解，即$\phi(p) = p_1^{r_1}p_2^{r_2}\cdots p_k^{r_k}$，若恒成立：
			\[g^{\frac{p - 1}{g}} \neq 1 \pmod{p}\]
				那么$g$就是$p$的原根。
		\item 若模$m$有原根，那么它一共有$\Phi(\Phi(m))$个原根。
	\end{enumerate}
